# Sakarie Hussein Ali — Portfolio

Personal portfolio built with Next.js 15, Three.js, and Framer Motion.

## Stack

- **Framework** — Next.js 15 (App Router, React Server Components)
- **3D / Animation** — Three.js, @react-three/fiber, Framer Motion
- **Styling** — Tailwind CSS, custom CSS variables (Deep Navy & Kinetic theme)
- **Language** — TypeScript

## Sections

1. **Hero** — Kinetic title with interactive Three.js mesh that responds to mouse movement
2. **ARIA Guide** — Animated SVG motion character that reacts to scroll depth
3. **Project Archaeology** — Interactive case studies: GraphRAG, CNN-XAI, Secure File Transfer, Distributed Orchestrator
4. **Technical DNA** — 3D rotating tech cloud, filterable by category
5. **Proof of Depth** — Research papers and live security scan animations
6. **Terminal** — Functional CLI (`ls projects`, `cat about`, `contact --send`, `help`)

## Dev

```bash
npm install
npm run dev
```
